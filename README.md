# 🎓 Student Roster Program

A simple command-line student roster program written in Python.

This program allows the user to:

- Add a student
- Remove a student
- Search for a student

---

## 📂 File Name

```text
student_roster.py
```

---

## 📌 Features

| Feature | Description |
|---|---|
| Add student | Add a student with name, email address, and registration date |
| Remove student | Remove a student by name |
| Search student | Search for a student by name |

# 🔤 Word Count Program

A simple command-line Python program to count word frequency from a text paragraph entered by the user.

The program will:

- Ask the user to enter a text paragraph
- Convert the text to lowercase
- Extract words from the text
- Count how many times each word appears
- Display the result in frequency order

---

## 📂 File Name

```text
word_count.py
```

---

## 📌 Features

| Feature | Description |
|---|---|
| User input | User enters a text paragraph |
| Lowercase conversion | Treats `Python` and `python` as the same word |
| Word extraction | Extracts words and ignores punctuation |
| Word frequency count | Counts how many times each word appears |
| Sorted output | Displays words in frequency order |

# ✅ Todo List Program

A simple command-line todo list program written in Python.

This program allows the user to:

- Add a new task
- Complete a task
- List all tasks
- Save tasks to a file
- Load existing tasks from the file when the program starts

---

## 📂 File Name

```text
todo_list.py
```

The todo list data will be saved in:

```text
todo_list.json
```

---

## 📌 Features

| Feature | Description |
|---|---|
| Add task | Add a new task to the todo list |
| Complete task | Mark an existing task as completed |
| List tasks | Display all tasks with their status |
| Save to file | Save tasks to a JSON file |
| Load from file | Read existing tasks when the program starts |

# 🧮 Calculator Program

A simple command-line calculator written in Python.

This program allows the user to enter two numbers and one operator, then calculates the result after validating the input.

---

## 📌 Features

- Accepts two numbers from the user
- Supports four basic operators:
  - `+` addition
  - `-` subtraction
  - `*` multiplication
  - `/` division
- Checks whether the number inputs are valid
- Checks whether the operator is valid
- Handles division by zero
- Uses functions to keep the code clean and reusable

---

## 📂 File Name

```text
calculator.py
```
